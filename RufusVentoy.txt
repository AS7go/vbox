=== Rufus и Ventoy — программы для создания загрузочных флешек. 
Rufus идеально подходит для создания одной загрузочной флешки с одной ОС. 
Ventoy — для мультизагрузочной флешки с несколькими образами.

- Описание и установка Rufus в Windows
Rufus — простая и быстрая программа для создания загрузочных USB-накопителей.

1.Перейдите на официальный сайт rufus.ie и скачайте последнюю версию.
https://rufus.ie 
Для большинства современных ПК подойдёт rufus-4.9.exe. Если у вас старый компьютер, скачайте версию с x86 в названии.

2.Вставьте флешку в компьютер и запустите Rufus.

3.В окне Rufus нажмите "Выбрать" и укажите путь к вашему загрузочному ISO-образу.

4.В поле "Схема раздела" выберите MBR, а в "Целевая система" — BIOS (или UEFI-CSM). 
Это сделает флешку совместимой с большинством компьютеров (MBR и UEFI).

5.Нажмите "Старт". Rufus отформатирует флешку и запишет на неё образ. По завершении флешка готова к использованию.

- Описание и установка Ventoy в Windows
Ventoy позволяет создать мультизагрузочную флешку, на которую можно просто копировать ISO-файлы.

1.Перейдите на официальный сайт ventoy.net и скачайте версию для Windows (файл с расширением .zip).
https://www.ventoy.net/en/download.html

2.Установите Ventoy. Распакуйте архив, вставьте флешку и запустите Ventoy2Disk.exe.

3.В программе выберите ваше USB-устройство и нажмите "Install". Ventoy подготовит флешку. 
Это нужно сделать только один раз.

4.После установки просто перетащите нужные ISO-файлы на флешку. Ventoy автоматически создаст меню загрузки.

5.Вы можете добавлять и удалять образы, как обычные файлы, без необходимости переформатирования флешки.

= Сравнение Ventoy и Rufus
Rufus — одноразовый инструмент: одна флешка = один образ. Он дает больше настроек, что полезно для старых систем.

Ventoy — универсальный инструмент: одна флешка = много образов. После установки достаточно просто копировать ISO-файлы. 
Это экономит время и идеально подходит для тех, кто часто работает с разными ОС.
